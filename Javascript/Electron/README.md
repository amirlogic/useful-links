# Electron

---


## Installation

```bash
npm install --save-dev electron@latest
```


## Electron Forge is the new way to go

```bash
npm init electron-app@latest my-app --template=webpack
```

Then to run the dev version:

```bash
npm start
```

Then to build:

```bash
npm run make
```
