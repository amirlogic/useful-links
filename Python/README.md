## Datatypes (most used)

Simple: String, Float, Integer, Boolean

Composed:

Lists: ```[ "val1", "val2", "val3" ]```

Dictionaries: 

```
{ "key1":"val1", "key2":"val2", "key3":"val3" }
```

Sets (unmutable):

```
{ "val1", "val2", "val3" }
```

Tuples (unmutable): ```( "val1", "val2", "val3" )```

Range: ```range(10)```

To get a variable type: ```type(var)```


# Casting

To integer: ```int()```

To float: ```float()```

To string: ```str()```


# String functions

Slicing: ```variable[a:b]``` a = start, b = end (not included)
